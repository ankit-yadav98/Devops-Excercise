## Demo Project - Cloud & IAAS Basics

Create server and deploy application on AWS.

### Technologies Used:
AWS, Linux, Java, Gradle

### Project Description:

- Setup and configure a server on AWS (Deploying EC2 Instance)
- Create and configure a new linux user on EC2 Instance (Security best practice)
- Deploy and run a Java Gradle application on EC2 server
- Access EC2 instance from local Desktop using RSA key pair.
